# Olá! Seja bem-vindo(a)! 🙋🏻‍♀️

Sou desenvolvedora de software com foco em back-end, atuando principalmente com Java, Spring Boot e Spring Data JPA

Tenho experiência prática no desenvolvimento de APIs REST, modelagem de dados com PostgreSQL e escrita de testes automatizados com JUnit e Mockito. Também possuo vivência com Docker, versionamento com Git e integração com front-end em Angular e TypeScript.

Gosto de entender como as soluções funcionam além da implementação, buscando escrever código organizado, testável e com atenção às regras de negócio.

Atualmente, sigo aprofundando meus conhecimentos em back-end e arquitetura, além de desenvolver projetos próprios para consolidar e expandir minha experiência prática.
